Using the Inject Method


September 18, 2014

Summary

The inject method in Ruby is commonly used as a way to accumulate a collection of objects. Given a set of objects, it will iterate over each one, executing whatever code is specified, and return a result. Inject is useful for dealing with collections of objects, such as strings, arrays, and hashes. Let's have a look at the format, and then I will go over some possible use cases.

Format

[array elements].inject(parameter) { |result, element| code to be executed}

The inject method is called in the same way as any other method. That is, with the dot method following the object. For this example I've used an array.

It's possible to pass arguments to the method. The argument passed will act as the starting point for the iteration. It could be an empty array, an empty hash, or a number.

Next we set up the code. The argument on the left is the result. It will always hold the value of the accumulated result of the all of the iterations of the code block. The argument on the right represents each element to be passed through the code block.

Uses

One of the simplest uses of the inject method is to return a sum of an array of numbers. For example:

[1, 2, 3, 4].inject(0) { |result, element| result + element }

In this case, we are iterating over the array [1,2,3,4] is added to the result for a total of 10. Each time we execute the code, the result is stored.

We could also use inject to build a hash based on some existing values stored in a multiple array:

petHash = [["cat", "sneaky"], ["dog", "great"], ["ferret", "stinky"]].inject({}){
|hash, element|
hash[element[0]] = element[1] hash
}

with the result:

{"cat"=>"sneaky", "dog"=>"great", "ferret"=>"stinky"}

In this case, we have iterated over an existing multidimensional array and used its values to create a new hash, called petHash. The parameter of the inject method is an empty array. The accumulated result is a hash. With each iteration, each element of the original array becomes a key-value pair in the new hash.
