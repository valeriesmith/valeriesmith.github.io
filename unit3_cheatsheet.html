Key Term Description Example
SELECT Used to select fields within a database SELECT name FROM grocery_items; banana, apple, tomato soup, granola bars
SELECT DISTINCT Used to list only different values from a given column SELECT DISTINCT aisle_type FROM grocery_items; produce, canned goods, snacks
*note that snacks is not repeated
WHERE Used to specify conditions to be met SELECT name FROM grocery_items WHERE type = 'produce'; banana, apple
OR Boolean comparison to check whether one condition is true SELECT name FROM grocery_items WHERE type = 'produce' OR type = 'snacks' banana, apple, granola bars
AND Boolean comparison to check whether both conditions are true SELECT name FROM grocery_items WHERE price > 1 AND price < 5 apple, tomato soup
ORDER BY Used to sort results. Default is ascending order SELECT name FROM grocery_items ORDER BY price ASC banana, apple, tomato soup, granola bars
Wildcards / LIKE Used to find all results containing characters specified. Used in conjunction with LIKE SELECT name FROM grocery_items WHERE name LIKE 'a%' apple
IN Used to find multiple values in a WHERE clause SELECT name from grocery_items WHERE name IN ('apple','banana') apple, banana
JOIN Used to combine results from multiple tables SELECT name, aisle_number FROM grocery_items JOIN aisles ON (type = aisle_type) apple => 1, banana => 1, tomato soup => 2, granola bar => 3
ALTER Used to change columns within a table ALTER TABLE grocery_items ADD sell_by DATE NOT NULL This will add a column with a sell by date for each item. This value must be filled in
SUM Aggregate function used to return the sum of the numeric column SELECT SUM(price) FROM grocery_items 11
COUNT Aggregate function used to return the number of rows that meet certain conditions SELECT COUNT(type) FROM grocery_items WHERE type = 'produce' 2
GROUP BY Used in conjunction with aggregate functions to group results SELECT COUNT(type), type FROM grocery_items GROUP BY type canned goods => 1, produce => 2, snacks => 1
