What is Pair Programming?

Pair programming is the practice of collaborating on programming tasks / projects between 2 developers.
Collaborating in this way has been proven to be very effective both to accelerate the task / project and the learning process of the developers.
The following are the main types of pair programming:

  • Driver / Navigator
  • Ping Pong
  • Novice / Expert
